Windows11部署Stable diffusion安装教程

发表于2023-08-12|更新于2023-08-16|AI
|字数总计:894|阅读时长:3分钟|阅读量:

前言

Stable-Diffusion：一个基于Latent Diffusion Models（潜在扩散模型，LDMs）的文图生成（text-to-image）模型。GitHub开源地址：https://github.com/Stability-AI/stablediffusion

stable-diffusion-webui：基于 Stable Diffusion 的基础应用，利用 gradio 搭建出的交互程序。 我们需要部署的就是stable-diffusion-webui。GitHub开源地址：https://github.com/AUTOMATIC1111/stable-diffusion-webui

配置软件运行的必要环境

  • Python 3.10.6（安装时注意勾选 Add Python 3.10 to PATH）
  • Git

安装 stable-diffusion-webui

从 Github 下载 stable-diffusion-webui 压缩包，随便解压到一个文件夹中，双击 webui-user.bat 文件运行。

注意：如果显示下载失败的话，需要设置一下代理，因为终端默认是不走代理的，使用 Clash 开启TUN Mode 即可，同时代理的网络要稳定，不然也会下载失败。

安装简体中文语言包

  1. 打开浏览器访问终端上显示的地址 http://127.0.0.1:7860
  2. 找到 Install from URL 选项，粘贴中文语言包的地址【https://github.com/VinsonLaro/stable-diffusion-webui-chinese 】之后点击 Install
  3. 点击下方的黄色按钮”Install”即可完成安装，然后重启WebUI(点击”Install from URL”左方的”Installed”，然后点击黄色按钮”Apply and restart UI”网页下方的”Reload UI”完成重启)
  4. 点击”Settings”，左侧点击”User interface”界面，在界面里最下方的”Localization (requires restart)“，选择”Chinese-All”或者”Chinese-English”
  5. 点击界面最上方的黄色按钮”Apply settings”，再点击右侧的”Reload UI”即可完成汉化

安装 ControlNet 扩展

找到“扩展插件”选项，粘贴 ControlNet 插件的地址（https://github.com/Mikubill/sd-webui-controlnet），

点击”安装“，插件安装完成后需要重启，找到“已安装”选项，点击“应用并重启用户界面”。

下载 ControlNet 模型

将下载好的文件保存到 /stable-diffusion-webui/extensions/sd-webui-controlnet/models 文件夹中。

不同模型对应的检测方式：

线稿检测 control_sd15_canny.pth
深度检测 control_sd15_depth.pth
模糊检测 control_sd15_hed.pth
直线检测 control_sd15_mlsd.pth
法线贴图 control_sd15_normal.pth
人体姿势检测 control_sd15_openpose.pth
涂鸦 control_sd15_scribble.pth
分块检测 control_sd15_seg.pth

下载训练好的模型

模型可以从 CivitAI 下载，不同的模型生成的图片类型不同，选择自己喜欢的类型下载即可。

生成图片

  1. 从最上方的”Stable Diffusion 模型(ckpt) “中选择需要的模型，从 CivitAI 中选择一张模型对应的图片，点击”Copy Generation Data“复制提示词和生成数据

  2. 将复制的提示词和生成数据粘贴到第一个提示词输入框中，点击右侧的”小箭头“可以自动填充数据。

  3. 调整一下宽度和高度，显卡性能较差的不建议设置的很大，不然生成速度会比较慢，生成批次可以设置一次生成的图片数量。

  4. 展开“ControlNet”选项，勾选“启用”，预处理选择“人体姿势检测”，模型选择 control_sd15_openpose，最后点击”生成“就可以了。

注意：第一次生成需要下载一些文件，速度会比较慢。
